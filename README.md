# HTML Source Viewer (Java)

## Overview

This is a simple Java GUI application that allows users to retrieve and display the HTML source code of a website.

## Functionality

By entering a URL, the application fetches the corresponding webpage and displays its raw HTML content.

## Purpose

The tool was created as a small experiment to work with HTTP requests and basic GUI development in Java.

## Status

This project is no longer actively maintained and remains available for historical and reference purposes only.
